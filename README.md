# 売買利益計算ツール

売買時(メルカリ等の出品前)に、売値・原価・送料から
利益と利益率を計算する簡易ツールです。

## 使い方
python revenue.py 売値 原価 送料
例
python revenue.py 5000 3000 500

## 出力内容
・利益
・利益率(原価基準/売価基準)
・赤字・低利益の判定

## 出力例(コマンドライン)
価格: 5000
原価: 3000
送料: 500
手数料: 500
利益: 1000
利益率(原価基準): 33 %
利益率(売価基準): 20 %
判定: 出品候補です
CSV出力完了:output/yyyymmdd_HHMMSS.csv

## 出力例(csvファイル)
価格,原価,送料,手数料,利益,利益率(原価基準),利益率(売価基準),判定,日時
5000,3000,500,500,1000,33,20,出品候補です,yyyy-mm-dd_HH:MM:SS

## なぜこのツールを作ったか（下書き）
・仕入れ前に毎回「本当に利益出るか？」を計算するのが面倒だった
・感覚で仕入れて失敗するのを防ぎたかった
・Pythonの練習を「実用」に寄せたかった

## このツールで解決したこと
・売値・原価・送料を入れるだけで利益と利益率が分かる
・赤字・低利益を即判定できる
・出品判断を感情ではなく数字でできる
